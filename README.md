# NBA Kill

NBA 主题的身份制卡牌对战 Web 游戏。4–8 人本地对局（人类 + AI），核心 / 队友 / 对手 / 独狼四方阵营，全套卡牌、装备、技能均围绕 NBA 球星与篮球场景重写。

🌐 **公网体验**：https://chaodewen.github.io/nba_kill/

## 快速开始

```bash
cd engine
npm install
npm run dev    # http://localhost:3005
```

构建：

```bash
cd engine
npm run build  # 产物输出到 engine/dist
```

## 设计要点

### 身份制
4–8 人对局生成核心 / 队友 / 对手 / 独狼组合（5–8 人 mapping 见 `Game.generateIdentities`）。**身份在玩家间随机分配**：人类不再永远是核心。核心身份的玩家额外 +1 体能上限。

> 默认对手身份对玩家隐藏（仅自己和阵亡玩家可见）。可通过 `🐞 调试` 按钮临时显示所有身份用于排查。

### 球员位置（同组协同）
三种位置取代单一阵营：
- **内线 (inside)** — 同位置队友存活时受到伤害 -1
- **后卫 (guard)** — 同位置队友存活时摸牌阶段额外摸 1 张
- **锋线 (forward)** — 同位置队友存活时投造成伤害后额外弃目标 1 张

### 节奏
所有玩家动作和响应统一 **2s** 展示间隔（`continueAfterCard` PACE_MULTIPLIER = 5）；常规事件日志按 1.2s 节奏排队弹出，避免连环刷屏。

### UI 布局
上方对手沿弧线分布（视觉相邻 = 引擎距离 1，对面 = 最远距离）；下方左侧是人类玩家信息卡，右侧是手牌 + 快捷按钮（投/佳得乐/弃/结）。日志面板独立于右侧。

## 操作
- 点击手牌使用；可用目标会高亮橙色光圈
- 出牌阶段需要选目标的牌（投、单挑、抢断、迫使失误、犯规麻烦、体能危机、做球）会进入目标选择模式
- 攻击时鼠标悬到合法目标会显示 MOBA 风格红色十字光标
- 弃牌阶段超出上限时弹模态由玩家自己挑要弃哪几张
- 头像 / 名字 / 技能行 / 装备 都可点击查看详情弹窗
